# DecodeLabs Internship - AI Chatbot Project

## Overview

This project is a simple rule-based AI chatbot developed as part of the DecodeLabs Artificial Intelligence Internship.

The chatbot interacts with users and provides information related to:

* Product details
* Payment methods
* Shipping information
* Order status
* Help menu

## Features

* Greeting users
* Displaying available products
* Payment information support
* Shipping information support
* Order status checking
* Help command
* Exit command

## Technologies Used

* Python
* Visual Studio Code
* Git & GitHub

## How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run:

```python
python chatbot.py
```

## Sample Commands

* hello
* products
* payment
* shipping
* order status
* help
* bye
